# 3D Water Bottle Design – SolidWorks

A sleek, ergonomic, and production-ready 3D model of a water bottle, designed entirely from scratch using **Dassault Systèmes SolidWorks**. This project showcases advanced parametric modeling, surface optimization, and assembly design.

## 🚀 Project Overview
This repository contains the complete design files for a modern water bottle. The design focuses on both aesthetics and functionality—balancing a comfortable ergonomic grip with precise mechanical features like leak-proof threading and a snap-fit cap assembly.

## 🛠️ Features & Design Techniques Used
*   **Parametric Modeling:** Built using robust design intents, allowing easy modification of dimensions (height, volume, diameter).
*   **Revolve & Shell Operations:** Utilized advanced revolve features for the main body and thin-walled shelling to ensure consistent wall thickness.
*   **Swept Boss/Base:** Applied to create precise, realistic threads on the bottle neck and cap for a secure, leak-proof seal.
*   **Ergonomic Grip:** Integrated surface fillets and cutouts for an enhanced, comfortable hand grip.
*   **Multi-Body Assembly:** Includes a fully functional assembly combining the bottle body, threaded cap, and rubber gasket seal.
